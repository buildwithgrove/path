### 1. Clone the repository

```bash
git clone https://github.com/buildwithgrove/path.git
cd ./path
```

### 2. Install all dependencies

Required tools:

```bash
make install_tools
```

Optional but highly recommended tools:

```bash
make install_tools_optional
```
